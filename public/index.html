IPO Result of Multiple BOID at Once

Looking for a way to check IPO result of multiple Users at once?

Here you can check IPO results of mulitple users at once. You need to simply provide the boid of each users in the input field below.

How does this system works?
First whenever a user visited this page, we send a request to https://iporesult.cdsc.com.np/ get the list of companies.

Once you fill the boid, we validate it, we send the boids and company id to our backend. Then on the backend, we make request to iporesult.cdsc.com.np for each boid. And send back the result.

iporesult.cdsc.com.np

iporesult.cdsc.com.np is a website by CDS and Clearing Limited. It was first released on Feb 27, 2021 (BS 2077/11/15)

In that website, you can select the company and provide your BOID number to check the result of your IPO. Before this, we have to login to meroshare, view the results but because of the huge number of requests meroshare used to be unresponsive.

A member, handels all the IPO related stuff of a family, thus if we can have a way to check the IPO of all members at once it would be easy. Thus we decided to develop this.
